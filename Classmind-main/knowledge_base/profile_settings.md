# Profile Settings

## What it does
Allows users to update their personal details, profile photos, display names, platform language, visual themes, and external cloud integrations.

## Why it exists
It provides a personalized workspace, localization controls, and account security management.

## When it should be used
Accessed anytime from the top-right user menu.

## How to use it
1. Click on your profile photo in the top right and select 'Profile Settings'.
2. Update fields like Name, Email, Theme, and Language.
3. Link your Google Drive account for automatic report exports.
4. Click 'Save Changes' to update profiles.

## Best practices
Upload a professional profile photo so students can identify their teacher easily in the Waiting Room and Chat.

## Common mistakes
Forgetting to click 'Save Changes' before navigating away from the settings panel.

## Troubleshooting steps
If Google Drive authentication fails, verify your browser is not blocking popups or check client ID settings.

## Related features
Theme Settings, Language Settings, Google Drive Integration
