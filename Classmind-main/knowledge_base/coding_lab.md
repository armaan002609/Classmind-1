# Coding Lab

## What it does
Embedded programming sandbox supporting multi-language compilation (Python, JavaScript, Java, C++, C, Go) with real-time test case verification.

## Why it exists
It enables computer science teachers to assign programming problems and automatically verify student code correctness.

## When it should be used
Used when teaching programming, algorithm design, or software engineering concepts.

## How to use it
1. Create a coding task with problem statement, starter code, and test cases.
2. Push the task to students.
3. Students write code in the embedded editor, compile, and run.
4. Code is auto-graded against hidden/visible test cases, and results are reported to the teacher.

## Best practices
Provide clear starter code and define robust test cases checking both basic logic and edge cases.

## Common mistakes
Not checking language syntax in the starter code, which causes student compilation errors immediately.

## Troubleshooting steps
If student code fails to run, check sandbox server resources or ensure the sandbox service is running on the backend.

## Related features
Tasks, Student Dashboard
