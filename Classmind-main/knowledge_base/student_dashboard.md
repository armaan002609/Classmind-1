# Student Dashboard

## What it does
Provides a clean, real-time portal for students to receive active tasks, write and run code in the coding lab, submit responses, view the leaderboard, raise hands, and chat.

## Why it exists
It creates an interactive, focused learning environment for students to participate in live classroom activities.

## When it should be used
Used by students during live class sessions to interact and submit work.

## How to use it
1. Go to the student login portal.
2. Enter the 6-digit session code provided by the teacher and enter your name.
3. Click 'Join as Student' and wait in the Waiting Room if the session has not started yet.
4. When live, view and complete tasks as they are sent by the teacher.

## Best practices
Use a desktop or laptop browser when working on coding tasks to have optimal editor space.

## Common mistakes
Closing the browser tab or navigating away during a locked test, which triggers cheating warnings on the teacher's dashboard.

## Troubleshooting steps
If tasks do not appear, ensure you are not suspended or check the WebSocket status at the bottom of the screen.

## Related features
Coding Lab, Tests, Tasks, Chat & Doubt System
