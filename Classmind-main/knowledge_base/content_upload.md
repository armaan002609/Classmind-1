# Content Upload

## What it does
Enables teachers to upload classroom documents, spreadsheets, slides, and images, automatically triggering AI metadata generation (objectives, summaries).

## Why it exists
It simplifies content distribution and structures document metadata automatically.

## When it should be used
Used when preparing files in the Content Hub.

## How to use it
1. Open the Content Hub.
2. Drag and drop file attachments or click 'Upload File'.
3. Wait for the server to process the file and generate learning objectives.
4. View files on student portals.

## Best practices
Upload PDFs and images instead of raw text files for better formatting in student portals.

## Common mistakes
Uploading unsupported file formats, which can result in metadata parsing failures.

## Troubleshooting steps
If upload fails, check if the file size exceeds server limits or verify write permissions in the data folder.

## Related features
Content Hub, Google Drive Integration
