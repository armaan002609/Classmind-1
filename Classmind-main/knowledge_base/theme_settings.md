# Theme Settings

## What it does
Provides visual style customizers (Dark Mode, Light Mode, System Theme matching) and custom color accents to adjust the VYOM UI aesthetic.

## Why it exists
It allows teachers and students to customize their workspace for comfort, accessibility, and visual preferences.

## When it should be used
Accessed from profile settings or the theme selector widget.

## How to use it
1. Open Profile Settings or click the theme icon on the header.
2. Choose between 'Dark', 'Light', or 'System' theme.
3. Select a color accent (e.g. VYOM Orange, Deep Violet, Royal Blue).
4. UI colors will transition immediately.

## Best practices
Use Dark theme in low-light environments to reduce eye strain, and choose high-contrast accents for accessibility.

## Common mistakes
Selecting low-contrast color combinations that make text hard to read.

## Troubleshooting steps
If the theme does not persist after reloading, check if cookies or localStorage are disabled in your browser.
