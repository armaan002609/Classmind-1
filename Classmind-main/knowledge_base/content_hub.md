# Content Hub

## What it does
Acts as a repository for classroom learning materials, enabling teachers to upload documents, slide decks, and code files which are instantly made available to students.

## Why it exists
It centralizes files sharing so students can view relevant documents directly inside their student portal.

## When it should be used
Used before or during class to distribute readings, worksheets, or guidelines.

## How to use it
1. Navigate to the Content Hub tab.
2. Drag and drop your files (PDF, images, text, json) or click to upload.
3. The platform will automatically generate smart learning objectives and descriptions for each file.
4. Students can view or download files from their content dashboard.

## Best practices
Upload worksheets and slides before starting the live session to minimize interruptions.

## Common mistakes
Uploading massive files (over 50MB) that can slow down loading times for students on slower connections.

## Troubleshooting steps
If a file is not visible to students, ensure the file is successfully uploaded and check its metadata status.

## Related features
Content Upload, Google Drive Integration
