# Teacher Dashboard

## What it does
Serves as the main control center for instructors to manage active class sessions, push questions, view attendance in real-time, monitor student understanding, and review raised doubts.

## Why it exists
It centralizes live class interactions, enabling teachers to quickly adjust their instruction and keep students engaged without shuffling between multiple tools.

## When it should be used
Used continuously during a live class session, as well as before starting a class to prepare materials.

## How to use it
1. Log into your VYOM teacher account.
2. Click 'Create Class Session' and enter a topic and duration.
3. Share the 6-digit session code with students.
4. Once students join, use the main dashboard panels to send tasks, resolve doubts, and view real-time participation metrics.

## Best practices
Keep the dashboard open on a secondary monitor if possible so you can monitor student status while presenting slides or code.

## Common mistakes
Forgetting to officially end the session using the 'End Class' button, which keeps the session status open and postpones reports compilation.

## Troubleshooting steps
If metrics are not updating, verify your internet connection or check the WebSocket connection indicator at the top of the dashboard. Refresh if disconnected.

## Related features
Session Management, Tasks, Attendance, Chat & Doubt System
