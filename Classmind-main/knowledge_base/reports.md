# Session Reports

## What it does
Generates comprehensive, professional reports summarizing class performance, individual student scores, attendance logs, and question analysis.

## Why it exists
It provides formal records that can be shared with school administration, parents, or saved for grading purposes.

## When it should be used
Generated automatically after a session ends.

## How to use it
1. Navigate to the Reports section.
2. Choose a completed session from the list.
3. Review the overview metrics (average score, pass rate, active time).
4. Click 'Download PDF' or 'Export Excel' to download the file.

## Best practices
Set up Google Drive integration so reports are automatically synced to your cloud drive upon session end.

## Common mistakes
Trying to download reports of active sessions; the session must be ended to complete reporting calculations.

## Troubleshooting steps
If PDF report generation fails, ensure the ReportLab library is working on your server.
