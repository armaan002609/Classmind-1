# Tasks Management

## What it does
Enables creation, scheduling, editing, and distribution of learning tasks (MCQs, Short Answer, Long Answer, Coding Challenges) to the classroom.

## Why it exists
It acts as the primary tool for active learning, turning passive lecturing into interactive training.

## When it should be used
Used before class to queue questions and during class to distribute them.

## How to use it
1. Navigate to the Tasks tab.
2. Click 'Create Task' and choose the question type.
3. Enter details, options, correct answers, and hints.
4. Click 'Assign Task' to push it immediately, or save it to the session queue.

## Best practices
Prepare a mixture of easy MCQs for checks-for-understanding and harder coding/short-answer challenges for group work.

## Common mistakes
Forgetting to set a correct answer for MCQs, which breaks automated grading.

## Troubleshooting steps
If students complain they cannot see a task, ensure the task state is set to 'Active' or 'Sent' and not 'Draft'.
