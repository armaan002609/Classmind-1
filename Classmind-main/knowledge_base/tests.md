# Tests Mode

## What it does
Initiates a formal, timed assessment environment where browser tabs are monitored, anti-cheat limits are active, and navigation is restricted.

## Why it exists
It provides a secure, high-integrity environment for classroom examinations and quizzes.

## When it should be used
Used for graded exams, weekly quizzes, or formal evaluations.

## How to use it
1. Go to the Tests tab.
2. Set up the questions, duration, and cheating tolerance thresholds.
3. Click 'Start Test'.
4. Monitor live student progress and cheat warnings in real-time. Click 'End Test' to conclude and lock all inputs.

## Best practices
Warn students before the test about the anti-cheat warning triggers to minimize accidental tab switching.

## Common mistakes
Leaving tests open indefinitely without setting an automatic time limit.

## Troubleshooting steps
If a student is locked out due to cheat warnings, the teacher can unlock them manually from the student control panel.
