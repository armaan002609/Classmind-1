# Notification System

## What it does
Dispatches real-time alerts, emails, and platform toasts to teachers and students regarding session actions, hand raises, cheat warnings, and report syncs.

## Why it exists
It keeps all classroom participants informed of important events instantly.

## When it should be used
Runs in the background during active sessions.

## How to use it
1. Go to Profile Settings -> Notifications.
2. Toggle email or desktop notifications for different categories (e.g. Cheat Alerts, Doubt Submissions, Attendance Updates).
3. Save configuration preferences.

## Best practices
Keep cheat alerts enabled on your desktop so you are immediately notified if a student leaves a test.

## Common mistakes
Disabling all notifications, leading to missed doubt submissions and student hand raises.

## Troubleshooting steps
If email notifications are not arriving, check your spam folder or verify SMTP credentials in the server configurations.
